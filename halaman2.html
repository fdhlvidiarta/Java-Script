Halaman 2

Ini adalah konten dari Halaman 2. Konten ini juga dimuat secara asynchronous menggunakan AJAX dan jQuery.

jQuery adalah sebuah library JavaScript yang mempermudah pengembangan aplikasi web dengan menyediakan berbagai fungsi dan metode yang dapat digunakan untuk manipulasi DOM (Document Object Model), efek animasi, penanganan event, dan juga untuk melakukan AJAX.
